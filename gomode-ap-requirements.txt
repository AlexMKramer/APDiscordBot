#   - kivy / kivymd : GUI, never needed to build world logic

# venv exactly (only Super Smash Bros. 64 x2 and Neon White fail to load, for non-dependency
# reasons). If a future seed uses a world that needs more, it degrades to "unsupported" and you
# add the dependency here.
#
